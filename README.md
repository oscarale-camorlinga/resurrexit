# resurrexit
A multi-language songbook and toolset for psalms for the Neocatechumenal Way.

Current features:
	- List of songs with verses and chords
	- View psalms by step or category
	- Change between languages

Current languages:
	- English (US)
	- Spanish (Mexico)
